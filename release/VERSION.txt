多平台影片下載器 V1.74
====================

發布日期: 2025-08-20
狀態: 穩定版本

主要修正:
1. 修正下載進度條顯示位置 - 確保只在「下載進度」標籤頁顯示
2. 優化下載完成狀態顯示 - 增加檔案大小和完成時間顯示
3. 避免重複下載 - 清除已完成項目後不會重複顯示連結
4. 改進下載速度 - 減少UI更新頻率，降低系統負擔
5. 修復標籤頁索引記憶功能
6. 增強下載完成通知 - 新增檔案開啟與資料夾開啟功能

系統需求:
- Windows 10/11
- 無需安裝 Python 或其他依賴

使用方法:
1. 雙擊執行 多平台影片下載器.exe
2. 首次運行會自動創建設定檔
3. 在「下載」頁籤中貼上影片連結開始下載
4. 切換到「下載進度」頁籤查看下載狀態

支援平台:
- YouTube
- TikTok / 抖音
- Facebook
- Instagram
- Bilibili
- X (Twitter)
- Threads
